📖 Overview

DocEase is a modern full-stack healthcare platform that enables patients to book appointments with doctors seamlessly. The platform provides dedicated dashboards for Patients, Doctors, and Administrators, ensuring efficient appointment scheduling, doctor management, secure authentication, and online payments.

Built using the MERN Stack, DocEase focuses on delivering a smooth and responsive healthcare experience with role-based access control and real-time appointment management.
